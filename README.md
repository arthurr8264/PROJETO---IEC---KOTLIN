Este é um breve guia sobre a linguagem **Kotlin**! Este repositório foi criado como parte da 2ª Verificação de Aprendizagem da disciplina de Introdução à Engenharia da Computação.

---

## 📌 O que é o Kotlin?
Kotlin é uma linguagem de programação moderna, multiplataforma e estaticamente tipada, criada pela JetBrains. Ela foi desenhada para ser concisa, segura e 100% interoperável com o Java, rodando perfeitamente na Máquina Virtual Java.

## 🚀 Principais Áreas de Aplicação
* **Desenvolvimento Mobile (Android):** O Google declarou o Kotlin como a linguagem principal e preferida para o desenvolvimento de aplicativos Android.
* **Desenvolvimento Back-end:** Muito utilizada na construção de servidores e APIs, especialmente com frameworks como o Spring Boot.
* **Desenvolvimento Multiplataforma (KMP):** Permite compartilhar a mesma lógica de código entre iOS, Android, Desktop e Web.

## ✅ Prós
* **Sintaxe Concisa:** Você escreve muito menos código se comparado ao Java tradicional, ficando visualmente mais "limpa".
* **Null Safety (Segurança contra Nulos):** O sistema de tipos do Kotlin foi feito para eliminar o perigo das exceções de referência nula (`NullPointerException`), que travam muitos aplicativos.
* **Interoperabilidade:** Você pode usar bibliotecas Java dentro de um projeto Kotlin e vice-versa, sem dor de cabeça.

## ❌ Contras
* **Tempo de Compilação:** Em alguns cenários de compilações limpas, pode ser um pouco mais lento que o Java puro.
* **Curva de Aprendizado:** Embora o básico seja fácil, conceitos mais avançados exigem um pouco mais de tempo de estudo.
